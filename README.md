# The Widget Factory

A toy project for experimenting with workflows in the BPMN world


## Stuff used
* SpingBoot 4.0.3
* Vaadin 25.0.3
* Camunda 7.0.24 (Legacy)

## Features
* Dynamic form-rendering based on user-tasks with form-keys

## Usage
`gradlew bootRun`

* http://localhost:8080 - The Widget Factory
* http://localhost:8080/camunda - Camunda cockpit (demo/demo)
